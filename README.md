# Hotel Market Analysis 

##  Project Overview

This project analyzes hotel data to understand pricing trends, customer preferences, and demand across major cities.

## Tools Used

* Python (Pandas, BeautifulSoup)
* Data Visualization

##  Key Steps

* Collected hotel data from MakeMyTrip using web scraping
* Cleaned dataset and handled missing values using Pandas
* Performed exploratory data analysis (EDA)
* Created visualizations to understand pricing and review trends

##  Key Insights

* Hotel prices vary significantly based on location and ratings
* High-demand cities show higher price fluctuations
* Customer ratings influence pricing patterns
